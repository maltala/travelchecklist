# TravelChecklist

Proyecto de práctica guiada con Django, Git y GitHub.

## Objetivo
Construir una aplicación web sencilla para gestionar viajes y tareas asociadas.

## Funcionalidades previstas
- página de inicio
- registro, login y logout
- CRUD de viajes
- CRUD de tareas por viaje

## Tecnologías
- Python
- Django
- SQLite
- Git / GitHub

## Cómo ejecutar el proyecto
> Este apartado se irá completando durante la práctica.
